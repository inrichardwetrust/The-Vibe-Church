大師不僅不二話，他還經常扔下一句話後說：「怎麼理解，自己慢慢悟。」

「自己慢慢悟」，就是要求你調用你本地的算力（大腦/樹莓派），並借助外部 GPU（AI 大模型），花上幾個小時甚至幾天去解壓縮。
