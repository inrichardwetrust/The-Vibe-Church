- **字源：** 災也。
    
- **玄樞解碼：**
    
    - 為什麼會有災？因為 **「各（外人）」** 進來了。
        
    - 為什麼外人能進來？因為你的防禦（口）有漏洞。
        
    - **「咎」** 不是天災，是 **「人禍」**。是因為你（人）沒有守好你的領地（口），導致敵人（夂）長驅直入。
