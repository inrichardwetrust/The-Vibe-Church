1. 直觀法：股是一個月加殳，殳看起來像是一個人跪著，下面的手在打他。或者是一個手拿著長鞭，或武器在擊打。股指自胯至膝的肉，引申為臀部。腓是月加非，非是兩個人相背，生氣了，背過身不見你，不太明白為何這樣設計。或是指，這塊肉中間鼓囊囊，下面細，和非的形狀很像？腓指小腿肚的肉。
2. 意會能量：都是腿上的肉，腿是爆發力，耐力，古人打獵需要大量使用。是跑車的轱轆。動力元件，人本身的自驅力。感覺有一種能量的儲備和釋放。
3. 受者視角：被支配，被驅使的力量，股和腓只能跟隨大腿骨的擺動，被動提供動力。是強力的執行單位。像牛。
4. 施者視角：腿是人最有力且攻擊范圍大的部位，用它打擊范圍廣力度大，純肉搏就是主力輸出。
5. 局部視角：股這部分肉似乎很適合用來懲罰。打幾十大板，讓人痛不欲生，卻仍能康復，不喪失勞動力。打屁股竹筍炒肉是小孩子必然要經歷的（西方近幾十年才開始不讓打孩子，中國依然是提倡，並未禁止。）
6. 隱藏屬性：可靠。沒有股，人就無法站立，行走。是人的支撐力量，股肱之臣。默默承受。
7. 全面視角：股可靠，承受，儲能；腓敏捷，強健，爆發。人走路、攻擊的重要工具。股是被懲罰的對象，寧可受皮肉之苦，也不要被砍手砍腳，只要還能留在牌桌上，都可以重來。股掌之上，股就是被操控的對象。驅使它，讓它承受傷害，為身體提供行動力和儲能。
