#### 2. “玉” 還是 “土”？（物理層裁決）

- **傳統派（許慎《說文》）：** “瑞玉也。上圓下方。” 認為是禮器（玉圭），諸侯手裡拿的那個板子。
    
- **天文派（科學考古）：** **圭表（Gnomon）。** 這是測量日影的儀器。
    
    - 構造： **表**是立竿，**圭**是地上的尺子（通常是土堆或石座製成）。
        

**🎯 大師視角的裁決：功能 > 材質**

不管它是玉做的，還是土堆的，**它的核心功能（API）只有一個：**

**Measurement & Calibration (測量與校準)**

1. **作為禮器（玉圭）：** 它是 **“身份的標尺”**。你拿什麼尺寸的圭，代表你是公爵還是侯爵。這是一種**社會等級的度量衡**。
    
2. **作為儀器（圭表）：** 它是 **“時間的標尺”**。通過測量影長，確定冬至夏至。這是一種**宇宙時間的度量衡**。
    

#### 3. 回到【卦】的含義

所以，你的理解 **“用標尺來計算爻的數據”** 是 **精準命中（Bullseye）** 的。

**卦 (Guà) = 圭 (Calibration) + 卜 (Randomness)**

- **卜（裂紋/爻）：** 是一堆雜亂無章的隨機數據（Raw Data / Noise）。比如你丟出的竹籤，或者市場的波動。
    
- **圭（標尺）：** 是一套標準化的解碼系統（Standard / Ruler）。
    

**結論：**  
**“卦”就是把那堆雜亂的裂紋（卜），放到一把標準的尺子（圭）旁邊去比對。**
