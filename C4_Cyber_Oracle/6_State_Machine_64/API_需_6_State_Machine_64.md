3. **受者視角：** 需要等待，等待时机。天下大势，顺势而为，如冲浪弄潮，势头未到，强冲动力不足。借势而上，等待上升气流。

5. **局部視角：** 需，内心空洞。缺失意义，缺失动力，缺失刺激。人需要不断的多巴胺刺激，如低血糖，一旦缺失就会无聊，内心煎熬，拿出手机刷短视频，嗑糖，填补空虚。
6. **隱藏屬性：** 需是积雨云，天上已经聚集了下落的势能，但是缺少凝结核，时机未到，因缘具足即可化作倾盆大雨。在等待期广积粮，时机一到乘风而起。
