龍，觀甲骨金文，是簡單的頭上有利角的蛇形生物。後來左邊強調龍頭形態，六書通裡變形為“辛、帝”，最後繁體字變為“立 + 月”，上面是鋒利的角，下面是張開的大口，右邊是捲曲的蛇身和利爪。
是非常強大的生物。
六書通裡有很多看起來像正面的龍的形象，好像龜。

最近看到中國各地爆出一些風暴中龍的錄像，在雲層裡龍形若隱若現飛來飛去，伴有龍吟。天有異象。龍也許是純能量體幻化。
所以有兩種龍的存在，一種是實體的巨獸，可能是古代的巨鱷、恐龍。一種是神話中的靈獸，能召喚元素魔法，上天入地。

龍是一種純陽的形象，龍鳳呈祥。
中國人崇尚龍，認為是龍的傳人、真龍天子，皇帝都是龍族。龍有通天神力，呼風喚雨。掌管風調雨順。
龍能屈能伸，大丈夫當像龍一樣潛龍在淵，見龍在田，飛龍在天，亢龍有悔。
中國龍文化與西方不同，中國龍通常是威嚴、正義，皇權，神明。西方龍多是為害一方的貪財邪惡噴火惡龍，騎士屠龍、國王屠龍成為美談。
