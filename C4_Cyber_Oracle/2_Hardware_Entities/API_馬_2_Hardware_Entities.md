反观于马，马是用来征战、劳动、共生的。

马，这个字用甲骨文写出来才发现古人对眼睛有多执着。为什么放大马的眼呢？是因为它有灵性吗？相反，在支配马时，人反而会遮住马的眼睛。冲锋时不会害怕，拉磨时不会厌烦。
马对于古代人有着绝对顶级的社会价值。军事上就是坦克，战车，国之重器，千乘之国就是横扫千军的存在。游牧民族无敌也就是得益于马。据说西班牙几百人灭印加古国就是带了一些马，把印加人吓住了，以为是什么神兽。
马在战争中给人带来绝对的不平等优势。有马无马是两个概念。在《骑马与砍杀》中有马的我可以单挑一个军团。蒙古人就是利用马的机动骑射战术征战天下。

马生命力强，现代很多疫苗制备也是靠马产生抗体再打给人。
马性格刚烈，能御马之人通常也是威风凛凛，如拿破仑骑马跨越阿尔卑斯山。
我听说马能感知人的性格，如果是懦弱之人，马不会服从，会把人甩下去；
如果是有威严有经验的人，马语者，烈马也会服服帖帖。

用马，需要因地制宜，战马留在冲锋时用，负重留给驮马，耕作留给挽马。
