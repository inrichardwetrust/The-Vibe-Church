另一方面，野豬在古代是非常危險的動物。危險程度僅次於老虎。我吃過野豬肉，非常粗糙腥味重。甚至有說法，家豬若流浪在野外，很快也會退化成野豬，很危險。憨厚的外表下，是一個能吃人的怪獸。
我姓朱屬豬，是極為特殊的命格。也許是一種集體潛意識，屬豬的人通常都比較快樂，幸福。易滿足。樂呵呵。豁達。享福的命。豬可以在廁所生活，豬打泥，艱苦的條件也能苦中作樂。這種能屈能伸的特性是他們種族延續到今天的關鍵，在人類社會的夾縫中拾人牙慧生存，是一個獨特的生態位。
豬象征著財富，小豬存錢罐。但是該用錢的時候會砸爛的。大師講過一個商業故事。智者不會辛苦打獵，而是默默編織柵欄，逐漸引誘野豬進入。等野豬警醒，已經身處柵欄內了。
作為人的啟示，要學會豬的隨遇而安，知足常樂。但不可陷入沉淪，縱情享樂。養肥了會被殺的。人的目的是找到回家的路，我們是誰，我們從哪裡來，我們到哪裡去。不能像豬一樣不能抬頭望天。覺醒，飛升。
