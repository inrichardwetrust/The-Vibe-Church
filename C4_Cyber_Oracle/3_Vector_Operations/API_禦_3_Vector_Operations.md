4. 御，使馬也。駕馭馬匹。乘是一種向上的動能，御是一種一馬平川向前的動能。似乎對應著平原區，守成，或平穩的發展趨勢。生住異滅中的住。
5. 御，皇帝專用的，御駕親征，御膳房，御醫。奮斗期已過，開始物質享受。
6. 禦，防禦。有了成績，有了財富，外人至。巨鱷巨鯊都想分一杯羹，行星想要用引力捕獲，隕。如何防禦？一是祭祀，我有如此強大的時間寶石不用嗎？示，祭祀。二是增強實力，借用保護傘力量，政治聯姻，祀與戎，我提供祀，保護傘提供戎。雙星系統。用高高的城墻讓心懷叵測者望而卻步。名將鎮守的邊疆無戰事，因為無人敢侵犯。不戰才是最好的防禦。
7. 禦之後就是生住異滅的異。故鼎上經常雕刻饕餮紋，警示自己勿貪婪。一馬平川的擴張期之後會陷入停滯，係統內部腐化，外人的覬覦。需自省。
